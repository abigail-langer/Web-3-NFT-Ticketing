### 测试步骤

1. **购买门票**
   ```
   访问: http://localhost:3000
   选择活动 → Buy Ticket → 选择座位 → 支付 0.01 ETH
   ```

2. **查看门票**
   ```
   点击 My Tickets → 查看门票列表
   ```

3. **生成二维码**
   ```
   点击"📱 显示二维码" → 保存二维码图片
   ```

4. **检票（组织者）**
   ```
   点击"🎫 检票系统" → 开始扫描 → 扫描二维码
   ```

5. **领取 POAP**
   ```
   检票后 → 点击"🎁 Claim POAP" → 确认交易
   ```

6. **查看 POAP**
   ```
   点击"🎖️ My POAPs" → 查看 POAP 列表
   ```

### 注意事项
- 检票功能需要摄像头权限
- 只有活动组织者可以检票
- 检票后才能领取 POAP
- POAP 持有者可以提前购票和选择优先座位
